# 感觉博物馆 · Museum of Feelings

一座在线博物馆，收藏各种各样的感觉。每个展厅都是一台可以上手玩的模拟器。

**在线参观：https://maxi-max-dev.github.io/museum-of-feelings/**

## 展厅

| 编号 | 展厅 | 状态 | 一句话 |
|---|---|---|---|
| №001 | 👁️ 眼睛商店 | 开放中 | 换一双眼睛，看同一个世界：近视、散光、色盲、猫、狗、蜜蜂、出生七天的婴儿 |
| №002 | 🚗 堵车制造机 | 开放中 | 一脚刹车如何变成一场幽灵堵车 |
| №003 | 📟 九键时代 | 开放中 | 用真的九键拼音打出「我想你」，计时 |
| №004 | 🚂 绿皮火车 | 开放中 | 哐当声、汽笛、泡面雾气和倒退的风景 |
| №005 | 🕸️ 算法织布机 | 开放中 | 你来当推荐算法，把信息茧房织成一块布 |
| №006 | 🏦 排队学 | 开放中 | 超市式对银行式，同一批人排两种队 |

## 逛展提示

- 眼睛商店里有几副眼睛藏着人眼看不见的东西，蜜蜂那副记得看花，猫那副记得入夜。
- 「用你的眼睛」按钮会请求摄像头权限，画面只在你的设备上处理，不会上传。

## 技术说明

纯静态页面，每个展厅一个单文件 HTML，零依赖零构建，下载下来双击也能开（只有摄像头模式需要 https 环境）。视觉效果为科学近似演示，供体验与科普，不作医学参考。
