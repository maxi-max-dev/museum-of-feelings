叫声 0 次 捕获 0 只

点一下洞穴（或按空格）= 发出一声叫声；移动鼠标 / 手指拖动 = 飞

说明牌 · 一帧一帧的声音照片
蝙蝠每秒发出十几到两百次超声，用回声的延迟算距离（一毫秒的来回，等于十七厘米），用左右耳收到的时间差算方向，用音调的变化算对方的速度。它的世界不是连续的画面，是一帧一帧的声音照片，每一帧都要自己叫一声才会显影。
说明牌 · 你玩的是慢放一万倍的版本
真实的蝙蝠声波每秒飞出去三百四十米，一次捕猎要修正航线几十次，快到飞蛾只听得见死亡逼近的滴答声。有些飞蛾进化出了反制：一听到蝙蝠切换成连发的锁定音，立刻收起翅膀，自由落体装死。
说明牌 · 人类同款
B 超用回声看胎儿，声呐用回声看海底。更神奇的是真实存在的人类回声定位者：一些盲人练出了用舌头打响声走路、骑车、打篮球的本事，大脑把原本处理视觉的那块皮层，直接改造成了另一种「看见」。
黑不黑不重要。对会听的耳朵来说，世界一直在回答。
